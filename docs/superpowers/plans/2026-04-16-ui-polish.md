### Task 3: Make Chorus Card Sticky (Replace Pop-in Bar)

**Files:**
- Modify: `src/pages/SongDetail.tsx:89-139`
- Modify: `src/styles.css:497-550`

The current implementation uses an `IntersectionObserver` to detect when the chorus scrolls out of view, then conditionally renders a separate `.chorus-sticky-bar`. This causes a jarring pop-in. Instead, make the actual `.chorus-card-wrapper` sticky so it naturally pins below the header when scrolled to.

- [ ] **Step 1: Simplify SongDetail.tsx — remove IntersectionObserver and sticky bar**

In `src/pages/SongDetail.tsx`:

1. Remove the `chorusStuck` state (line 24): delete `const [chorusStuck, setChorusStuck] = useState(false)`
2. Remove the `chorusRef` ref (line 28): delete `const chorusRef = useRef<HTMLDivElement>(null)`
3. Remove the entire IntersectionObserver `useEffect` (lines 45-56)
4. Remove the sticky chorus bar JSX (lines 102-108):
   ```tsx
   {/* DELETE THIS BLOCK */}
   {chorusStanza && chorusStuck && (
     <div className="chorus-sticky-bar" style={{ fontSize }}>
       <div className="stanza-label">{chorusStanza.label}</div>
       {chorusStanza.text}
     </div>
   )}
   ```
5. Remove the `firstChorusIdx` variable and `chorusStanza` variable (lines 89-90)
6. Remove the `ref={i === firstChorusIdx ? chorusRef : undefined}` from the chorus stanza rendering (line 123)
7. Simplify the stanza rendering — no need to track first chorus anymore

The updated stanza rendering (lines 118-139) becomes:

```tsx
{song.stanzas.map((stanza, i) => {
  if (stanza.is_chorus) {
    return (
      <div key={i} className="chorus-card-wrapper">
        <div className="chorus-badge">{stanza.label || 'Chorus'}</div>
        <div className="chorus-card">
          <div className="stanza-text">{stanza.text}</div>
        </div>
      </div>
    )
  }
  return (
    <div key={i} className="verse-card">
      <div className="stanza-label">{stanza.label}</div>
      <div className="stanza-text">{stanza.text}</div>
    </div>
  )
})}
```

The full updated component imports become:
```tsx
import { useState, useEffect, useRef, useLayoutEffect } from 'react'
```
(Same as before — `useRef` and `useLayoutEffect` are still used for the header height measurement.)
