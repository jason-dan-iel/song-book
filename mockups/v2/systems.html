How to read this: each column is one complete design system rendered at mobile width. Scroll within a column to see Home → Category → Song detail. All three use Fraunces + Tiro Devanagari Hindi — difference is alignment, ornament, rule treatment, accent weight.

System 1 · Editorial — current direction refined. Left-aligned, hairline rules, forest accent only on interaction, numeric indices in tnum. Default pick if you want a modern serif book.
System 2 · Hymnal — centered composition, Roman numerals, fleuron glyphs (❦ § ✦), burnt-sienna accent, italic kickers. Most literary, most ornamented.
System 3 · Breviary — 3px double-rule dividers, numbered entries prominent in big serif, tracked uppercase labels, tighter index feel. Most like an old hymnal plate.
