# Sample Text File Format for Song Lyrics
# =========================================
#
# INSTRUCTIONS:
# 1. Start with CATEGORY (hindi or english)
# 2. Then TITLE on next line
# 3. Then stanzas and chorus sections
# 4. Use "STANZA X:" for verse labels
# 5. Use "CHORUS:" for chorus sections
# 6. Leave blank lines between sections
# 7. Each line of lyrics goes on its own line
# 8. The script will auto-generate the song number based on existing files
#
# IMPORTANT: Save your text file with UTF-8 encoding to preserve Hindi characters!

CATEGORY: hindi
TITLE: तेरी महिमा अपरम्पार
